# 財經數據分析平台

基於 Python Flask 的財經數據分析網站，暗色主題，互動式圖表。

---

## 專案結構

```
財經數據分析平台/
├── app.py                        # Flask 主應用，定義所有 API 路由
├── data_service.py               # 資料存取層（目前讀 CSV，未來可換成 API）
├── requirements.txt              # Python 套件依賴
├── templates/
│   └── index.html                # 主頁面（頁籤式架構：大盤監控區 + 國際股市區）
├── static/
│   ├── css/style.css             # 暗色主題樣式
│   └── js/charts.js              # 所有 Plotly 圖表邏輯
├── private/
│   └── generate_stats.py         # 私下統計分布報告產生器（獨立執行）
└── All_Data/
    └── 日資料/
        ├── 大盤統計/
        │   └── 大盤統計資訊/     # 大盤統計 CSV（上市/上櫃/興櫃每日資料）
        │       └── *.csv
        ├── 國際股價指數/          # 國際股市指數 CSV
        │   └── *.csv
        └── 國內銀行利率(日)_國內銀行匯率/  # 各銀行每日匯率 CSV
            └── *.csv
```

---

## 資料來源設定

資料目錄定義於 `data_service.py` 上方：

```python
# 大盤統計
DATA_DIR       = Path(r"...\大盤統計\大盤統計資訊")
# 國際股市
INTL_INDEX_DIR = Path(r"...\國際股價指數")
FX_DIR         = Path(r"...\國內銀行利率(日)_國內銀行匯率")
```

- 系統啟動時自動讀取資料夾內**所有 CSV 檔案**。
- 若有重複的（證券代碼＋年月日）組合，以**檔名較新**的檔案為準。
- 新增 CSV 後，點擊網頁右上角「↺ 重新載入」按鈕，或呼叫 `/api/reload` 即可更新（同時清除大盤統計與國際股市快取）。

### CSV 格式要求

| 欄位 | 說明 |
|---|---|
| 證券代碼 | 標的識別碼（如 `Y99992 上市-股票`）|
| 年月日 | 格式 `YYYYMMDD`（如 `20260313`）|
| 成交金額 / 成交數量 / 成交筆數 | 每日交易量 |
| 總委買/委賣數量、筆數 | 委託單統計 |
| 漲停/跌停委買/委賣數量、筆數 | 停板委託統計 |
| 上漲/下跌/持平/未成交家數 | 每日家數統計 |
| 漲停/跌停家數 | 每日停板家數 |

> 編碼：UTF-16，分隔符：Tab（`\t`）

## 大盤監控區圖表說明

### 1. 成交量位階儀表板（3 個 Gauge）
- 三個市場（上市、上櫃、興櫃）各一個半圓儀表板。
- 顯示最新一日成交金額在**過去滾動一年**資料中的百分位數。
- 色區：極低量（深藍）→ 偏低（藍）→ 正常（灰）→ 偏高（橙）→ 極高（紅）。

### 2. 市場廣度熱力圖
- X 軸：交易日，Y 軸：三市場。
- 顏色：紅（上漲比例低）→ 綠（上漲比例高）。

### 3. 各市場漲跌家數比例折線圖（3 張）
- 每張圖對應一個市場，Y 軸為佔總家數比例（%）。
- 四條線：上漲%、下跌%、漲停%、跌停%。
- 點擊圖例可隱藏或顯示個別折線。
- 支援日期範圍篩選器拖曳調整。

### 4. 委買/委賣力道比
- 各市場每日（總委買數量 ÷ 總委賣數量）折線圖。
- 數值 > 1 表示買盤佔優，< 1 表示賣盤佔優。

### 5. 成交金額趨勢（含移動平均與歷史百分位帶）
- 日成交金額折線，疊加 MA5（黃色點線）、MA20（紫色虛線）。
- 背景色帶：歷史 P20 ～ P80 區間（依全期資料計算）。

### 6. 資金流向堆疊面積圖
- 三市場成交金額佔比隨時間演變（百分比堆疊）。

### 7. 市場廣度震盪指標（Breadth Oscillator）
- 計算：（上漲家數 − 下跌家數）÷ 總家數 × 100
- 柱狀圖，正值（綠）= 多頭主導，負值（紅）= 空頭主導。

### 8. 漲停/跌停比率
- 各市場每日漲停家數 ÷ 跌停家數折線圖。
- 比率越高代表市場情緒越亢奮；< 1 則偏向恐慌。

## 國際股市區功能說明

### 頁籤切換
點擊頂部導覽列「國際股市區」頁籤，首次進入時自動載入指數清單與預設圖表。

### 指數選擇（兩欄勾選清單）
- **主軸欄（左）**：勾選後指數數列顯示於圖表左 Y 軸。
- **副軸欄（右）**：勾選後指數數列顯示於圖表右 Y 軸（獨立刻度）。
- 同一指數只能在一欄勾選（兩欄互斥）。
- 搜尋框可即時按名稱或代碼過濾，兩欄同步篩選。
- 各欄右上角「清除」按鈕可一鍵取消本欄所有勾選。

### 日期設定
- **顯示期間（起～迄）**：決定折線圖 X 軸範圍。
- **基準日**：累積報酬率圖的起算基準點（走勢圖不受基準日影響）。
- 快速預設按鈕：近3月 / 近6月 / 近1年，同時設定顯示起始日與基準日。
- 點擊「套用」後重新載入並繪圖。

### 圖表說明

#### 指數走勢（原始數值）
- Y 軸：各指數原始報價數值（日經 225 ≈ 38,000、S&P 500 ≈ 5,800 等）。
- 因各指數規模差距大，請善用主軸（左 Y 軸）與副軸（右 Y 軸）分別顯示不同量級的指數。
- 可涵蓋貨幣：USD、JPY、EUR、GBP、AUD、CAD、HKD、KRW、SGD、ZAR、CHF、CNY、TWD、LOCAL（MSCI 本幣加權）。

#### 累積報酬率走勢
- Y 軸：自基準日起算的**台幣計價**累積報酬率（%）。
- **LOCAL 指數**：MSCI 本幣加權指數無對應匯率，**不列入本圖**。
- **TWD 指數（台灣）**：無匯率效果，顯示本幣指數報酬。
- **單一指數模式**（僅勾選 1 個非 LOCAL 指數）：顯示三條拆解線：
  - 實線：台幣計價總報酬 = (1 + 本幣報酬) × (1 + 匯率報酬) − 1
  - 虛線：本幣指數報酬（剔除匯率）
  - 點線（黃）：匯率貢獻（TWD/外幣，正值＝外幣升值，對台灣投資人有利）
- **多指數模式**：每個指數顯示一條台幣計價總報酬線（統一比較基準）。

### 座標軸刻度調整
每張圖下方各有一列控制欄：
- **主軸 / 副軸**：分別設定最小值、最大值與刻度間距。
- 「套用軸設定」：直接更新該圖軸範圍，**無需重新 fetch 資料**。
- 「重置」：恢復自動刻度。
- 留空表示自動縮放。

### 支援指數分組

| 分組 | 說明 |
|---|---|
| 台灣 | 台灣加權指數、電子加權、OTC |
| 亞洲 | 日本、香港、韓國、中國、新加坡、澳洲 |
| 歐洲 | 德國、法國、英國、荷蘭、義大利、瑞士、北歐等 |
| 美洲 | 美國（S&P 500、NASDAQ、Russell 2000）、加拿大 |
| MSCI | AC World、新興市場、EAFE 等各區域 Local / USD / EUR 版本 |
| 其他 | 南非等 |

## 私下統計分布報告

```bash
python private/generate_stats.py
```

執行後於 `private/stats_report.html` 產生報告，用瀏覽器直接開啟即可。

- 包含 21 個數值欄位，每欄位各做：
  - 直方圖（含中位數標示線）
  - Box plot（含平均數與標準差）
  - 統計摘要表（計數、均值、標準差、各百分位、偏態、峰態）
- 三個市場分欄並排比較。

## API 路由

### 大盤監控區

| 路由 | 說明 |
|---|---|
| `GET /` | 主頁面 |
| `GET /api/meta` | 資料元資訊（日期範圍、標的清單）|
| `GET /api/gauge` | 各標的成交金額百分位數 |
| `GET /api/timeseries?start=&end=` | 時序資料（折線圖等）|
| `GET /api/capital-flow?start=&end=` | 資金流向資料 |
| `GET /api/heatmap` | 市場廣度熱力圖資料 |
| `GET /api/reload` | 重新載入所有 CSV（含國際股市快取）|

### 國際股市區

| 路由 | 說明 |
|---|---|
| `GET /api/intl/indices` | 可用指數清單（含貨幣、分組、日期範圍）|
| `GET /api/intl/chart-data?codes=&start=&end=&base=` | 指定指數的原始數值、本幣／匯率／台幣累積報酬率 |

**`/api/intl/chart-data` 參數說明：**

| 參數 | 說明 | 範例 |
|---|---|---|
| `codes` | 指數代碼，可多次傳遞 | `codes=SB04&codes=SB22` |
| `start` | 顯示起始日 | `2025-01-01` |
| `end` | 顯示截止日 | `2026-03-13` |
| `base` | 基準日（報酬率起算點） | `2025-01-02` |

**回傳 JSON 欄位說明（每個指數）：**

| 欄位 | 說明 |
|---|---|
| `raw` | 指數原始數值 |
| `local_return` | 本幣累積報酬率（%） |
| `fx_return` | 匯率累積報酬率（%），正值＝外幣對台幣升值 |
| `twd_return` | 台幣計價總報酬率（%） |
| `can_decompose` | 是否有匯率資料可拆解（`LOCAL` / `TWD` 指數為 `false`）|
| `currency` | 計價貨幣代碼（`LOCAL` = MSCI 本幣加權，不列入報酬率圖）|
