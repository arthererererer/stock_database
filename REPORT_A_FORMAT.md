# 研究報告 a 格式參照

> 本文件作為撰寫其餘研究報告（b、c、d…）時的格式與架構參照，確保一致性。

---

## 一、報告結構模板

```
1. 解讀指南（rpt-interpret）
   - 議題說明、進場邏輯、公式定義
   - 各章節衡量議題與解讀方式

2. 主體內容（依議題而定）
   - 1.1 後續累積報酬分布（直方圖 + 統計表）
   - 1.2 後續超額報酬分布（直方圖 + 統計表）
   - 1.3 持續性/特徵分析（圖表 + 圖下統計表）
   - 1.4 分組比較（圖表 + 圖下統計表）
   - 1.5 特徵分析（各特徵箱型圖 + 圖下統計表）

3. 結論（rpt-section）
   - 整體觀點、個股層級高持續性解讀、市場層級、注意股與炒作風險

4. 方法說明（rpt-note）
   - 事件定義、進場邏輯、公式、篩選規則
```

---

## 二、進場與報酬公式

| 項目 | 定義 |
|------|------|
| **進場邏輯** | T 日收盤買進，T+h 日收盤賣出 |
| **累積報酬** | exp(Σ ln(1+r)) − 1，其中 r 為日報酬（小數） |
| **超額報酬** | (1+R_s)/(1+R_m) − 1，R_s=個股累積報酬，R_m=大盤累積報酬 |

---

## 三、圖表規範

### 3.1 箱型圖
- **僅顯示箱型**（boxpoints=False、boxmean=False），不顯示散點
- **不內嵌統計數據於圖中**
- **圖下獨立統計表**：N、Mean%、Median%、Std%、偏態、峰態、P10/P25/P75/P90、勝率

### 3.2 分組定義
- **1.5 股票特徵**：依各特徵**百分位數**三分組
  - 高組：> P75
  - 中組：P25 ~ P75
  - 低組：< P25
  - 適用：市值、本益比、股淨比、現金殖利率、CAPM Beta
- **1.3 / 2.3 持續性**：積分分數 ≥ P80 = 高持續性（約 20%），< P80 = 一般（約 80%）
  - 分析對象：**1.3 為所有振幅大事件**；**2.3 為所有振幅小事件**
  - 對振幅小事件：高持續性 = T+1~T+20 異常振幅積分高，即振幅自 T 日異常縮小後逐漸回升或放大
  - 圖表：① 高持續性事件 T~T+20 累積積分演化（中位數連線 + P25~P75 灰階面積）、② 一般事件 T~T+20 積分演化、③ 高 vs 一般 T+20 累積報酬箱型圖

### 3.3 圖例
- 避免冗餘圖例（如直方圖與箱型圖重複標示相同組別）
- 圖例置於圖下方，水平排列
- **1.3 積分演化圖**：橫軸 T~T+20，縱軸累積積分；中位數連線、P25~P75 灰階面積；高持續性與一般事件各一圖

---

## 四、解讀指南必含內容

1. **極端樣本說明**：若 T+20 出現尾端極值，建議以 Median、P25、P75 解讀，並於解讀指南簡述可能原因
2. **50%+ 報酬合理性**：厚尾分布、小型股/高 Beta 特性說明
3. **高低定義**：明確寫出分組門檻（中位數、P80 等）

## 七、議題 a 專用定義（可依議題調整）

| 定義 | 說明 |
|------|------|
| 振幅大事件 | 今日高低價差% > 前20日 rolling P90 |
| 振幅小事件 | 今日高低價差% < 前20日 rolling P10 |
| 積分分數 | Σ 異常振幅(T+k)，k=1~20 |
| 高持續性 | 積分 ≥ 所有事件 P80 |
| 累積積分演化 | 橫軸 T~T+20，縱軸 Σ 異常振幅(T+j)，j=1..k；中位數 + P25~P75 灰階 |
